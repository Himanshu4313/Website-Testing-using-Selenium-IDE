# Website-Testing-using-Selenium-IDE
website testing (Canva.com) using selenium IDE 

## Project Description

This project demonstrates functional testing of the Canva web application using Selenium IDE. 
Multiple test cases were designed and executed to verify core features of the website including login authentication, logout functionality, page loading, navigation between pages, search functionality, and creating a new design.

The project focuses on validating expected results against actual outputs to ensure correct application behavior.

## Tools & Technologies

- Selenium IDE
- Functional Testing
- Web Application Testing
- Browser Automation

 ## Test Scenarios

The following functionalities were tested:

1. Login verification with valid credentials
2. Logout functionality
3. Homepage loading validation
4. Navigation to templates page
5. Search functionality using search bar
6. Create a new design

## Sample Test Case

Test Case ID: Login_001  
Test Scenario: Verify login with valid credentials  

Steps:
1. Open Canva login page
2. Enter valid email
3. Enter verification code
4. Click on login button

Expected Result:
User should be successfully logged in and redirected to dashboard.
